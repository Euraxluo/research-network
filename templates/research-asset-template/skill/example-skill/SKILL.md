# Example Research Skill

## Purpose

This skill helps an Agent perform a specific research capability.

## Inputs

- Research problem
- Available datasets
- Constraints

## Workflow

1. Understand the problem.
2. Select method.
3. Generate code or analysis.
4. Validate result.
5. Produce paper-ready output.

## Output

- Analysis notes
- Code
- Figures
- Paper section

## Quality Gates

- Explain assumptions.
- Cite sources.
- Provide reproducibility steps.
