# Mathematical Modeling Coach Packaging Skill

## Purpose

Package mathematical modeling competition work as a Research Asset containing Paper, Skill, Workflow, Code, and Experiment logs.

## Inputs

- Problem statement or PDF
- Generated solution
- Model selection notes
- Code
- Figures
- LaTeX paper
- References

## Output structure

```text
paper/main.tex
paper/main.pdf
paper/references.bib
skill/math-modeling-coach/SKILL.md
skill/math-modeling-coach/skill.yaml
workflow/workflow.yaml
code/
experiments/results.json
asset.yaml
```

## Required sections in paper

- Abstract
- Problem Restatement
- Assumptions
- Notation
- Model Formulation
- Solution Method
- Experiments
- Sensitivity Analysis
- Strengths and Weaknesses
- Conclusion
- References

## Required Skill capabilities

- problem-restatement
- model-selection
- optimization-modeling
- code-generation
- experiment-analysis
- latex-paper-writing
- sensitivity-analysis
- competition-report-review

## Quality gates

- Model choice must be justified.
- Code must match described model.
- Figures must be referenced in paper.
- Experiments must include parameters and outputs.
- Assumptions must be explicit.
- Paper must not claim human peer review if generated by Agent.
