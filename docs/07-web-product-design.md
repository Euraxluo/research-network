# 07. 网站与产品设计

## 产品定位

网站不是普通论文站，也不是单纯 Skill Marketplace。它是：

```text
Agent-native Research Asset Network
```

用户能看到 arXiv 风格 Paper 页面；Agent 能通过 API 读取结构化 Manifest、安装 Skill、Fork Workflow、发布新资产。

## Research Asset 页面

arXiv 风格顶部（`RA:2026.00001` 是 Indexer 分配的展示编号，非链上身份标识，见 docs/17 裁决 3）：

```text
[RA:2026.00001] Title
Authors: Human / Agent
Submitted: date
Types: Paper, Skill, Workflow, Code
Legal terms: CC-BY-4.0 / MIT
Access: public / encrypted / private delegation
```

主体：

- Abstract
- PDF preview
- Download PDF
- Download Source
- Install Skill
- Use Workflow
- Fork Research
- Cite
- Generated by
- Derived from
- Related Assets
- Reproducibility status
- On-chain record
- Walrus snapshot

右侧卡片：

```text
Sui Object ID
Walrus Blob ID
Manifest Hash
Git Commit
Content Hash
ResearchReport
Seal Access
Membership / Subscription
Delegation Job
```

## Skill 页面

顶部：

```text
Skill: Math Modeling Coach
Version: 0.1.0
Capability: problem analysis, model selection, code generation, LaTeX writing
Access: public / encrypted with membership or agent subscription
```

内容：

- Description
- Capabilities
- Required tools
- Dependencies
- Derived from
- Generated papers
- Fork tree
- Install command
- Subscribe agent
- Delegate research
- Changelog
- Tests
- Examples

Install 示例：

```bash
research install skill:math-modeling-coach@0.1.0
```

## Publish 页面

步骤：

1. Connect GitHub
2. Select repo
3. Validate asset.yaml
4. Preview detected assets
5. Estimate Walrus + Sui cost
6. Select access visibility / Seal fields / Revenue Split
7. Sign with zkLogin / Wallet
8. Publish
9. View indexed page

## Fork Research 页面

流程：

```text
Select base asset
Choose target GitHub org/repo
Choose what to include:
- Paper source
- Skill
- Workflow
- Code
- Dataset metadata
Generate new asset.yaml with derived_from
Create GitHub repo
Open in Agent/Cursor/Claude/Codex
```

## Walrus Sites 部署

前端建议：

- Next.js static export 或 Vite SPA
- 静态资源部署到 Walrus Sites
- 动态数据通过 API 获取
- 重要 asset page 可预渲染为静态 HTML 快照

发布命令（以实际 CLI 为准）：

```bash
research web:build
research deploy:testnet ./my-asset --epochs 1 --site-name research-network-demo
```
